TIPOS DE DATOS EN PROBABILIDAD Y ESTADISTÍCA

Datos cuantitativos discretos

Concepto:

Son datos numéricos que resultan de un conteo y solo pueden tomar valores enteros específicos.

Características:

  • Son numéricos.
  • Son contables.
  • No admiten valores intermedios.

Ejemplo:

Una universidad analiza el número de intentos que realizan 100 estudiantes para aprobar un examen en línea de certificación en redes durante un semestre.

Datos cuantitativos continuos

Concepto:

Son datos numéricos que provienen de una medición y pueden tomar cualquier valor real dentro de un intervalo, incluyendo decimales infinitos.

Características:

  • Se miden, no se cuentan
  • Requieren intervalos para organizarse.
  • Admiten decimales.

Ejemplo:

Una empresa de desarrollo de software analiza el tiempo de respuesta de su servidor web (en segundos) durante 100 solicitudes consecutivas.

Datos cualitativos

Concepto:

Son datos que describen características o categorías, no cantidades numéricas.

Características:

  • Representan categorías.
  • Se analizan mediante proporciones o porcentajes.
  • Representan cualidades.

Ejemplo:

Una empresa analiza el sistema operativo utilizado por sus 100 empleados en sus equipos de trabajo.
